# FieldNotes AI

A public learning journal documenting real-time AI development — built by a non-developer using Claude Code.

🌐 [fieldnotes-ai.com](https://fieldnotes-ai.com)

## What this is

FieldNotes AI is where I document building with AI in public. Every session gets a field note — real costs, real mistakes, what actually happened. No highlight reels.

The site is aimed at practitioners and AI-curious developers who want honest accounts of what building with Claude Code actually looks like.

## Built with

- Next.js 14 App Router
- Contentful CMS (headless, 6 content types)
- Tailwind CSS
- Vercel (deployment)
- Claude Code (primary development tool)

## The meta part

This repo is itself a field note. Every commit corresponds to a documented session at fieldnotes-ai.com. The git history is the receipts.

## Running locally

```bash
npm install
cp .env.local.example .env.local
# Add your Contentful credentials to .env.local
npm run dev
```
